solana上的代币

  • 代币是代表对各种资产所有权的数字资产。 代币化使得财产权的数字化成为可能，是管理 可替代和不可替代资产的基本组成部分。
    • 可替代代币代表同类型和同价值的可互换和可分割资产（例如 USDC）。
    • 不可替代代币（NFT）代表不可分割资产的所有权（例如艺术品）。

SPL (Solana Program Library)

  • 代币程序（Token Program）包含与网络上的代币（包括可替代和不 可替代）交互的所有指令逻辑。
  • 铸造账户（Mint Account）代表一种特定类型的代币，并存储关于代 币的全局元数据，如总供应量和铸造权限（有权创建新代币单位的地址）。
  • 代币账户（Token Account）跟踪特定地址拥有的特定类型代币（铸 造账户）的单位数量。

SPL TOKEN

创建Mint Account 铸币账户

spl-token create-token

查看token

spl-token account-info –address FNUJxCihY9EVZBt5U3Vk6E1HTPhhjCFiViKmgSUabjyS(token地址)

spl-token create-account FNUJxCihY9EVZBt5U3Vk6E1HTPhhjCFiViKmgSUabjyS(token地址)

账户之间的关系

Mint Account 是“造币机”，
Token Account 是“钱包”，
关联代币账户（Associated Token Account, ATA）是“用户默认的钱包地址”。

Mint Account（铸币账户）

  • 作用： 定义一种 Token 的基本属性，比如：
    • 小数位数（decimals）
    • 谁是铸币人（mint authority）
    • Token 总供应量（supply）
    • 是否允许冻结（freeze authority）
  • 结构体：spl_token::state::Mint
  • 一个 Token 一个 Mint
  • **Mint 是造币机，它本身不接收 Token **

✅ 举例：USDC、SOL、LSY Token，每种 Token 都有一个独立的 Mint。

Token Account（代币账户）

  • 作用： 表示「某个地址持有某种 Token 的多少数量」。
  • 每个 Token Account 只能对应一个 Token（mint）。
  • 结构体：spl_token::state::Account
  • 你可以拥有多个 Token Account 来装同一个 Token。

✅ 举例：

  • 你有两个 USDC 钱包，分别存在不同 Token Account 中。
  • 也可以转账到别人的 Token Account，只要 mint 匹配就行。

关联代币账户（ATA：Associated Token Account）

  • 作用： 给你提供一个「固定格式」的钱包地址，专门用来接收某种 Token。
  • ATA 是 “用户对某个 mint（Token）” 的默认接收账户
  • 它是 PDA（程序派生地址），地址可以预测，规则如下：
  • 所以：
    • 每个用户对每种 Token 只有一个 ATA
    • 大多数 DApp 和钱包默认只支持 ATA

✅ 举例：

你是用户 A，想接收 LSY Token，你就自动有一个 ATA，别人转账会转到这个 ATA。

三者之间的关系图（类比现实生活）：

名称 比喻 作用
Mint Account 造币厂 定义一类 Token 的规则（如 USDC、LSY）
Token Account 多个钱包账户 用户存放某种 Token 的地址
ATA（关联账户） 默认主钱包地址 系统自动生成的“标准钱包地址”，只能有一个

举个通俗例子：

假设你发行了自己的 Token 叫做 LSY Coin：

  • 你创建了一个 Mint Account：专门负责管理 LSY 的铸造、销毁。
  • 张三想持有你的 LSY Coin，他需要一个 Token Account。
  • 为了方便，系统自动为张三生成了一个 ATA（其实就是一个 Token Account，但地址固定、自动生成）。

使用场景对照：

你要做的事 用到的账户类型
创建新 Token Mint Account
给用户发 Token 用户的 Token Account
自动获取用户接收地址 Associated Token Account (ATA)
查某人有多少某 Token 查他的 Token Account（通常是 ATA）

总结一句话：

Mint Account 定义了 Token 本身，Token Account 是存 Token 的钱包，ATA 是你默认的、固定格式的钱包地址。
